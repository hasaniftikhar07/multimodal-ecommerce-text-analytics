# Multi-Modal eCommerce Text Analytics Framework

This repository contains the implementation and experiments for a multi-modal NLP framework for analyzing eCommerce customer reviews.

The framework integrates:

- Emotion classification (transformer-based)
- Sentiment analysis (lexicon-based)
- Topic modeling (LDA)

## Overview

The goal of this project is to demonstrate how multiple NLP paradigms can be combined to extract deeper business insights from customer reviews.

Dataset size: 23,485 reviews

Modalities integrated:

- Emotion prediction
- Sentiment polarity scoring
- Topic extraction
- Cross-modal consistency analysis
